VERSETE:
GET  /api/verses                    ← Lista cu filtre + paginare

GET  /api/verses/random             ← Verset aleatoriu
GET  /api/verses/versetul-zilei     ← Verset consistent pe zi
GET  /api/verses/carti              ← Lista cărților cu statistici
GET  /api/verses/capitol/:ab/:cap   ← Capitol specific

GENERATOR:
GET  /api/generate/templates        ← 62 template-uri + custom
GET  /api/generate/teme             ← 12 teme disponibile
POST /api/generate                  ← Generează descriere + hashtags
POST /api/generate/upload           ← Upload template custom
DELETE /api/generate/templates/:f   ← Șterge template custom

POSTĂRI:
GET  /api/posts                     ← Lista postărilor
GET  /api/posts/:id                 ← Postare specifică
POST /api/posts                     ← Salvează postare
PUT  /api/posts/:id                 ← Actualizează postare
DELETE /api/posts/:id               ← Șterge postare
GET  /api/posts/stats/summary       ← Statistici postări

SOCIAL / FACEBOOK:
GET  /api/social/status             ← Status conectare FB
POST /api/social/publish-direct     ← Publică imediat
POST /api/social/schedule           ← Programează postare
GET  /api/social/scheduled          ← Lista programărilor
GET  /api/social/history            ← Istoric publicări
POST /api/social/publish/:id        ← Publică din DB
DELETE /api/social/scheduled/:id    ← Anulează programare

CITIRE BIBLIE:
GET  /api/reading/progress          ← Progres general
POST /api/reading/mark              ← Marchează capitol citit
GET  /api/reading/carte/:abrev      ← Progres per carte
GET  /api/reading/suggest           ← Sugestie zilnică

SISTEM:
GET  /health                        ← Health check

text

/ → redirect la /dashboard

/dashboard    ← Dashboard principal
              → Greeting cu nume Facebook
              → Verset aleatoriu (se schimbă la 30 min)
              → Progres citire Biblie
              → Stats: versete, publicate, programate, descrieri
              → Status Facebook live
              → Programări active cu countdown
              → Activitate recentă
              → Progres bars Biblie

/generate     ← Generator postări
              → Pasul 1: Alegere imagine (62+ template-uri)
              → Pasul 2: Alegere verset + temă + platformă
              → Pasul 3: Descriere + hashtags + publicare
              → Preview imagine cu verset + logo
              → Download PNG 1080×1350
              → Publică pe Facebook
              → Programează cu dată/oră

/schedule     ← Programări
              → Lista postărilor programate
              → Countdown până la publicare
              → Buton "Publică acum"
              → Buton "Șterge"
              → Istoric publicări cu link Facebook

/history      ← Istoric postări
              → Toate postările cu status
              → Filtrare: draft/scheduled/published/failed

/verses       ← Biblioteca biblică
              → 31.093 versete răsfoibile
              → Căutare full-text
              → Filtrare pe carte/testament
              → Statistici Biblie
              → Verset aleatoriu 🎲
              → Copiere verset

/settings     ← Setări
              → Status Facebook (valid/invalid)
              → Token Facebook
              → Teste publicare

GLOBAL:
              → Toggle Dark/Light theme ☀️🌙
              → Tema salvată în localStorage
              → Mobile responsive
